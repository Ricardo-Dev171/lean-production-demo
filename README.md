# Production Flow Control - Demo Vercel

Projeto React + Vite pronto para deploy no Vercel.

## Rodar localmente

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy no Vercel

1. Crie um repositório no GitHub.
2. Suba o conteúdo desta pasta.
3. Acesse https://vercel.com/new
4. Importe o repositório.
5. Clique em Deploy.

Framework preset: Vite
Build command: npm run build
Output directory: dist
